## v0.9.0

### Новое
- **`expect.status`** — можно указать список допустимых HTTP-кодов для каждого шага; если сервер ответил другим кодом, запрос считается ошибкой
- **`vars`** — глобальные переменные тест-плана, доступные всем виртуальным пользователям без повторного объявления
- **`--live`** — живой вывод статистики в лог каждую секунду: RPS, ошибки и полная latency (p50/p90/p95/p99, среднее, мин, макс)
- **Latency в таймлайне** — `timeline_1s` теперь включает перцентили latency за каждую секунду, с разбивкой по шагам

### Улучшения
- Улучшения производительности и кодовой базы

## v0.6.0

### Новое
- Docker-образ на базе `scratch` (~20 MB)
- `--cpu-monitor` — предупреждение при высокой загрузке CPU (Linux)
- `--log-vus` — логирование изменений числа активных VU

### Улучшения
- Per-VU transport по умолчанию: каждый VU работает с собственным пулом соединений
- `--shared-transport` для режима с общим пулом
